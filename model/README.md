# WebPPL model

## Installation

Install webppl

```
npm install -g webppl
```

Install [webppl-json](https://github.com/stuhlmueller/webppl-json) package

```
mkdir -p ~/.webppl
npm install --prefix ~/.webppl webppl-json
```

## Running

Run

```
webppl coordinate_DSL.wppl --require webppl-json
```
